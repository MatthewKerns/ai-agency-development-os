### **Step 3: Create Gamma Theme**

**Process:**
```
1. Open Gamma.app
2. Start new presentation
3. Input Business DNA elements:
   - Paste mission statement
   - Paste UVP
   - Paste key messages
   - Add brand colors
4. Let Gamma AI generate initial theme
5. Customize:
   - Adjust colors to brand palette
   - Set custom fonts
   - Add logo/brand assets
   - Create slide templates
6. Test with sample content
7. Save as iCodeMyBusiness theme
```

## Slide Template Details

### 1. Title Slide
- Large, bold headline
- Subtitle/tagline
- iCodeMyBusiness logo
- Presenter name (optional)

### 2. Section Header
- Section title
- Optional icon or graphic
- Minimal text
- Transition slide between topics

### 3. Text + Image (50/50)
- Equal split layout
- Text on left, image/graphic on right
- Use for case studies, examples

### 4. Bullet Points
- Maximum 5 bullets
- Large, readable font
- Icons for each bullet (optional)
- White space for clarity

### 5. Code Example Slide
- Syntax-highlighted code block
- Monospace font
- Explanation text above/below
- Before/After comparison option

### 6. Before/After Comparison
- Side-by-side layout
- Clear labels
- Visual distinction (color/arrows)
- Use for transformations, improvements

### 7. Process Diagram
- Step-by-step workflow
- Arrows connecting steps
- Icons for each step
- Clean, linear flow

### 8. Stats/Metrics
- Large numbers
- Supporting text
- Visual elements (charts, graphs)
- 2-4 key metrics per slide

### 9. Testimonial
- Quote in large text
- Client name/photo
- Company/role
- Optional logo

### 10. CTA Slide
- Clear call-to-action
- Contact information
- Website link
- QR code (optional)
