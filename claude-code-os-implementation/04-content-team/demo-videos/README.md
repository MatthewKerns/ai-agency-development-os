## Demo Standards

### Video Requirements
- **Length:** 2-5 minutes (respect viewer time)
- **Format:** Screen recording with voiceover
- **Quality:** 1080p minimum
- **Audio:** Clear narration, no background noise

### Structure
1. **Hook** (10 sec) - What problem does this solve?
2. **Overview** (30 sec) - High-level what you'll see
3. **Walkthrough** (2-4 min) - Show it working
4. **Results** (30 sec) - What's the outcome?
5. **CTA** (10 sec) - Next step for viewer

### Tools
- Screen recording: Loom, OBS, or QuickTime
- Editing: CapCut, DaVinci Resolve
- Hosting: YouTube (unlisted), Loom, Google Drive
