## Usage Tips

1. **Copy templates as needed** - Don't edit these templates directly
2. **Customize for your workflow** - Adapt sections as you learn what works
3. **Link liberally** - Connect related insights across files
4. **Update regularly** - Keep filling in outcomes and learnings
5. **Review monthly** - Update coach profiles and top insights
