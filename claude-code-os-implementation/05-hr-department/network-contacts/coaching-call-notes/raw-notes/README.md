# Raw Coaching Call Notes

**Purpose:** Store Fathom exports (summaries + transcripts) before processing

---

## How to Use This Directory

1. **After coaching call ends:**
   - Wait for Fathom to process (usually 2-5 minutes)
   - Open Fathom recording
   - Copy summary, topics, action items, and transcript

2. **Create new file:**
   - Format: `YYYY-MM-DD-coach-name.md`
   - Example: `2025-11-25-richard.md`
   - Use TEMPLATE.md structure

3. **Save Fathom export:**
   - Paste all Fathom content
   - Add your live notes from during call
   - Save file

4. **Process within 24 hours:**
   - Follow PROCESSING-WORKFLOW.md
   - Extract insights and action items
   - Add to by-coach and by-topic files
   - Add urgent items to daily planning

5. **Mark as processed:**
   - Rename file to `YYYY-MM-DD-coach-name-PROCESSED.md`
   - Keeps raw notes searchable
   - Easy to see what's been processed vs pending

---

## File Naming Convention

**Before processing:**
```
2025-11-25-richard.md
2025-11-20-denis-daigle.md
```

**After processing:**
```
2025-11-25-richard-PROCESSED.md
2025-11-20-denis-daigle-PROCESSED.md
```

---

## What to Include

✅ **Always include:**
- Fathom summary
- Fathom topic extraction
- Fathom action items
- Full transcript
- Your live notes during call
- Pre-call context (what you asked about)

❌ **Don't include:**
- Personal/sensitive client info (keep separate)
- Anything confidential shared by coach

---

## Processing Checklist

Each raw note should have this at the bottom:

```markdown
## Processing Checklist

- [ ] Extract key insights
- [ ] Categorize action items by urgency
- [ ] Add to by-coach/[coach-name].md
- [ ] Add insights to relevant by-topic files
- [ ] Create daily planning todos for urgent items
- [ ] Identify follow-up questions
- [ ] Mark as PROCESSED (rename file)

**Processed on:** [Date]
```
