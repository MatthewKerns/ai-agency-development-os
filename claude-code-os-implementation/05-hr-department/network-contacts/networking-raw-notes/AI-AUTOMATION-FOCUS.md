# AI/Automation Agency Contact Tracking Focus

## Why This Matters

For an AI automation agency, knowing someone builds "voice agents" or specializes in "n8n workflows" is MORE valuable than knowing they code in Python or React.

**Old approach (traditional dev agency):** Track tech stack (React, Node.js, Python)
**New approach (AI automation agency):** Track automation types and platforms

## Processing Contacts: What to Capture

When processing networking notes, ask these questions:

### For Developers/Builders:
1. **What automation types do they build?**
   - Voice agents? CRM? Email? RAG? Workflows?
2. **What platforms do they use?**
   - n8n? Make? Zapier? Claude Code? Custom?
3. **What's their specialization?**
   - "Voice agent specialist"
   - "CRM automation expert"
   - "n8n workflow builder"
4. **What specific projects have they built?**
   - Not generic "built a website" but "built voice agent for sales training"

### For Lead Generators:
1. **What automation problems do their clients have?**
   - Need CRM automation? Voice agents? Email systems?
2. **How do they get clients?**
   - Warm outreach? Events? Content? Referrals?
3. **What results do they get?**
   - "4 deals in 4 weeks" (Simon)
   - "¬£8k deal ‚Üí retainer" (Scott)
